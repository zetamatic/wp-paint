WP Paint - WordPress Image Editor is a browser based Image Editor for WordPress media images.

== Description ==

WP Paint - WordPress Image Editor is a browser based HTML5 Image Editor for WordPress media images. It has an intuitive interface resembling most common Desktop based Photo Editors with an extensive array of Image Editing, Photo Manipulation and Photo Editing Features.

= Features =

* Image editor with an intuitive desktop Photo Editor like interface
* Ability to Manage Layers
* Flip, Rotate or Resize images
* Add Custom Text to images
* Crop images
* Apply Blur effects to images
* Turn images to Grayscale
* Change Hue of images
* Adjust image brightness contrast
* Add shapes on top of images

== Installation ==

1. Login to your WordPress dashboard and navigate to Plugins > Add New
2. Search for "WP Paint".
3. Click "Install".
4. Click "Activate".
5. Once the plugin is activated, you can see "Edit Image using WP Paint" link next to the "Edit Image" link on all your media browsers.

= Can I edit the images like desktop Photo Editor? =
Yes, you can edit the images like a desktop Photo Editor, but with browser level limitations.

= Will this plugin work with my theme? =
Yes, absolutely the plugin should work with your theme without any issue.

= Will this plugin work with Gutenberg Editor? =
